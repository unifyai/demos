From this directory:


Create file for environment variables:
```
touch .env
```

Add the following to your new .env file:

```
OPENAI_API_KEY=<Your OpenAI API Key>
GMAIL_ADDRESS=<Your GMail Address>
GMAIL_PASSWORD=<Your GMail Password>
```

The set up the virtual environment:
```bash
uv venv .browser_use
source .browser_use/bin/activate
uv pip install -r requirements.txt
```

Install playwright:
```
playwright install
```

Run the demo:
```python
python main.py
```

Check your new "Browser Use" interface in [the console](https://console.unify.ai) to see the live logs.

See this link for more details on the demo this is derived from:
https://docs.browser-use.com/quickstart
